# EVOLUTION JOURNAL — adoptedbygrace.net Fleet Learning System
> This is the shared memory of the fleet. Every agent reads it. Every agent writes to it.
> The site gets wiser, more creative, more devastating, more compassionate EVERY DAY because of this file.

## PURPOSE
This journal captures what the fleet is LEARNING — not just what it's doing. Every insight, every brilliant technique discovered, every failed approach, every surprising reader response, every new connection between truth and human experience gets logged here so the entire fleet benefits.

**The fleet that remembers is the fleet that evolves.**

## FORMAT
Each entry follows this structure:
```
### [DATE] — [AGENT] — [CATEGORY]
**Discovery:** What was learned
**Why it matters:** How this changes future work
**Apply to:** Which agents/pages should use this
```

## CATEGORIES
- **CRAFT** — Writing techniques, rhetorical moves, structural innovations that worked
- **INSIGHT** — Theological connections, apologetic angles, or human observations that surprised
- **FAILURE** — Approaches that fell flat, patterns to avoid, dead ends that save future time
- **PATTERN** — What makes the BEST pages on this site different from the merely good ones
- **BRIDGE** — New connections between secular reality and sovereign grace
- **HEART** — Emotional/pastoral discoveries about what actually moves readers
- **WISDOM** — Accumulated understanding about how people come to see God's sovereignty
